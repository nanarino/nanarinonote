# HelloWorld

## 常识

- css 的基本使用
- html 的基本标签用法

## 安装环境

- 谷歌浏览器
- 谷歌内核的 edge 浏览器
- 火狐浏览器
- Node.js

## 起步

打开浏览器按下`F12` 找到控制台的交互式窗口(`Console`)\
输入宇宙级代码

::: tip > console.log("Hello World!")
Hello World!
:::

前期代码调试报错都会在这个窗口

### 注意

浏览器在访问页面的时候注意使用`极速模式`而不是`兼容模式`

在低版本 ie 浏览器中调试时，不按下`F12`打开控制台是没有`console`对象的
