# nanarinonote

自用学习笔记，基于 vuepress@1.7.1，包含

- JavaScript基础
- Python基础

## page-build

先度构建：<https://nanarino.github.io/nanarinonote/>

本地运行需将配置文件中的`base: '/nanarinonote/'`改为`base: '/'` 

## page-cli-setup

安装依赖

```bash
# nvm use 14.19.2+
npm install
```

脚手架命令：

```bash
# serve with hot reload at localhost:8080
npm run docs:dev

# build for production with minification
npm run docs:build
```
